04 · PROTOCOLO CRÍTICO

El puente JSON ⇄ doPost

Recuerda: Usa JSON para evitar el error de CORS.

Este ocurre cuando un navegador bloquea una petición web entre diferentes dominios por seguridad. Se soluciona configurando el servidor para que incluya las cabeceras Access-Control-Allow-Origin.

protocol.js 
fetch(URL_WEB_APP, {
  method: "POST",
  body: JSON.stringify({ fila: [...datos] })
}) // ✦ JSON salta CORS, sin headers prohibidos

La plantilla
vacía.

Cinco bloques. Si los entregás todos, no hay error de permisos, ni mantenimiento posterior, ni alucinación de diseño.

[IDENTIDAD]: usa estos logos [Links] y estos colores [Colores]. Estilo [Estilo].
[FLUJO]: Pantalla 1 → datos; Pantalla 2 → interacción; Pantalla 3 → feedback.
[CONECTOR]: enviar a [URL del Script] usando JSON (Recuerda: Usa JSON para evitar el error de CORS).
[LÓGICA]: "mi backend espera un JSON con la propiedad fila (un array)".
[MAPEO]: "dame los encabezados según el orden del array que generes".

Evita el error de permisos

JSON + ContentService = intercambio fluido. Cero CORS.

Mantenimiento cero

Cambias el HTML, el doPost no se toca. Solo "pega la fila".

● MASTERY
PhD · semiconductores 15 años

La tunelización cuántica deja de ser ruido despreciable cuando el gate dieléctrico baja de ~1.2 nm; ahí entran en juego materiales high-κ como HfO₂ y, más recientemente, estructuras gate-all-around (GAA) en nodos de 3 nm.

La implicación práctica: el escalamiento de Dennard ya no aplica linealmente. La industria migra a backside power delivery (PowerVia, BSPDN) y a fotolitografía EUV High-NA para sostener la curva. TSMC N2 y el A14 de Intel son los primeros nodos comerciales donde esto convive.

nivel: terminología técnica precisa ✓ experto
