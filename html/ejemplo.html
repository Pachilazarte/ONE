Bienvenido

Esta es una página de ejemplo que puedes utilizar como base para crear tus propios contenidos. Está completamente sincronizada con el sistema de Drive → GitHub → Netlify.

💡 Consejo: Edita el HTML de este archivo para personalizar el contenido según tus necesidades.

Características

  • ✅ Diseño responsivo (funciona en móvil y desktop)
  • ✅ Sincronización automática desde Google Drive
  • ✅ Hospedado en Netlify con CDN global
  • ✅ URLs limpias y amigables con SEO
  • ✅ Sistema de navegación integrado

¿Cómo usar este archivo?

Sube este archivo a la carpeta de Google Drive, ejecuta la sincronización en Google Apps Script, y automáticamente aparecerá en tu sitio web. Puedes hacer cambios en cualquier momento y sincronizar nuevamente para actualizar el contenido.

Ejemplos de Contenido

Puedes incluir cualquier tipo de contenido HTML:

  • Formularios interactivos
  • Encuestas y cuestionarios
  • Catálogos de productos
  • Calendarios y agendas
  • Dashboards y analytics
  • Videos embebidos
  • Mapas interactivos
📌 Importante: Los archivos HTML deben estar en formato UTF-8 para que caracteres especiales (acentos, ñ) se muestren correctamente.
